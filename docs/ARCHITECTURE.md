## Component Details

### Frontend (React + Vite)

| Component | Description |
|-----------|-------------|
| Dashboard | Unified risk overview with charts and metrics |
| Deepfake Detection | Image/video upload and analysis UI |
| Phishing Detection | URL scanning interface |
| AML Detection | Transaction analysis form |
| Alerts | Centralized alert management |
| Reports | Analytics and report generation |

**Tech Stack:**
- React 18 with Hooks
- Vite for bundling
- TailwindCSS for styling
- Framer Motion for animations
- Recharts for data visualization
- React Router for navigation
- Axios for API calls

### Backend (FastAPI)

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/v1/dashboard/stats` | GET | Dashboard statistics |
| `/api/v1/deepfake/analyze/image` | POST | Analyze image for deepfakes |
| `/api/v1/deepfake/analyze/video` | POST | Analyze video for deepfakes |
| `/api/v1/phishing/analyze` | POST | Analyze URL for phishing |
| `/api/v1/aml/analyze` | POST | Analyze transaction for AML |
| `/api/v1/alerts` | GET/POST | Alert management |

### ML Models

#### 1. Deepfake Detection
```
Model: EfficientNet-B0 + Xception Ensemble
Input: 224x224 RGB Image
Output: [0-1] Probability of being fake
Dataset: FaceForensics++
Accuracy: 94.2%
```

#### 2. Phishing Detection
```
Model: Random Forest Classifier
Input: 17 URL features
Output: [0-1] Phishing probability
Dataset: PhishTank + OpenPhish
Accuracy: 97.1%
```

#### 3. AML Detection
```
Model: Isolation Forest + GNN
Input: Transaction features
Output: Anomaly score + risk factors
Dataset: Synthetic + Real data
Accuracy: 89.5%
```

## Security Considerations

1. **Input Validation**: All user inputs sanitized
2. **File Upload**: Size limits, type validation
3. **CORS**: Restricted to known origins
4. **Rate Limiting**: Prevent API abuse
5. **Logging**: Comprehensive audit trail

## Scalability

- **Horizontal Scaling**: Stateless API design
- **Model Serving**: Can deploy to TorchServe/TensorFlow Serving
- **Caching**: Redis for frequently accessed data
- **Database**: PostgreSQL with read replicas
